Snake Game


Project Overview

This is a Snake game developed using C# and Object-Oriented Programming principles. It features a Nyan Cat-inspired visual, a dynamic leaderboard to track scores, and a two-player function for multiplayer gameplay.

Features

  • Dynamic gameplay with snake movement and collision detection
  • Nyan Cat-inspired visual theme for a fun experience
  • Leaderboard to track high scores of players
  • Two-player function for competitive gameplay

Technologies Used

Developed with C# using Visual Studio, the game is structured with Object-Oriented Programming principles for easy scalability and maintainability. The user interface (UI) is simple yet intuitive, allowing players to easily navigate and play the game.
